# 🎮 TileMania
## CS 4700: Game Design Studio

Build a complete, challenging 2D platformer — tile worlds, patrol enemies, projectile combat, health systems, and polished game feel — using **Unity 6 + C#**.

## 🗺️ What You'll Build

| Feature | Concepts Taught |
|---|---|
| Tile-based world with Rule Tiles | Tilemap system, auto-tiling, composite colliders |
| Player run / jump / climb | Rigidbody2D, ground detection, better jump physics |
| Sprite animation | Animator Controller, blend trees, transitions |
| Smart camera | Cinemachine, confiner, camera shake |
| Patrol enemies | AI state, ledge detection, FlipX |
| Arrow shooting | Projectile instantiation, trigger collision, fire rate |
| Health & lives | UI hearts, invincibility frames, singleton GameSession |
| Scene transitions | SceneManager, build index, DontDestroyOnLoad |
| Audio & polish | AudioManager singleton, particles, screenshake |

## 🎯 Learning Objectives

By completing TileMania you will be able to:
- Design and build multi-layer tile maps with Rule Tiles
- Implement physics-based player movement with polished jump feel
- Create state-machine-driven animations
- Write patrol AI that detects ledges and turns safely
- Build a projectile system with proper layer-based collision
- Implement a health and lives system with UI feedback
- Manage game state across multiple scenes using the Singleton pattern
- Apply game feel techniques: camera shake, invincibility frames, sound effects
